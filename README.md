বাংলাদেশ সময়ে FIFA বিশ্বকাপ ২০২৬-এর ফিক্সচার ট্র্যাকার।

## ফিচার
- ৭২টি গ্রুপ পর্যায়ের ম্যাচের সময়সূচি (বাংলাদেশ সময়)
- দল অনুযায়ী ফিল্টার
- স্কোর এন্ট্রি ও পয়েন্ট টেবিল অটো আপডেট
- শেয়ার বাটন
- ডার্ক/লাইট থিম
- বাংলা/ইংরেজি ভাষা টগল
- PWA — মোবাইলে ইনস্টল করা যাবে

## Deploy করার উপায়

### Vercel (সহজতম):
1. https://vercel.com এ অ্যাকাউন্ট খুলুন
2. GitHub-এ এই project push করুন
3. Vercel থেকে import করুন → Deploy

### Netlify:
1. https://netlify.com এ অ্যাকাউন্ট খুলুন
2. "Add new site" → "Deploy manually"
3. `npm run build` চালান, `dist` folder drag করুন

### Local চালানো:
```bash
npm install
npm run dev
```
